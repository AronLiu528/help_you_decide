# help_you_decide

A Side Project <br>

# [用途]
提供一個隨機抽獎，解決日常上難以決定的小問題！

# [功能]
1. 隨機產生一個選項。 <br>
2. 可編輯項目與選項列表。 <br>
3. 本地存儲保存數據。 <br>

# [使用工具]
1. Flutter/Dart。 <br>
2. GetX 狀態管理。 <br>
3. shared_preferences套件實現本地儲存。<br>
